## プロジェクト

### Gemini Canvas Slide Prompt Generator

Gemini用のスライド作成プロンプトを自動生成するツールです。

🌐 **オンラインで使う**: [https://yun-program.github.io/claude-code-playground/](https://yun-program.github.io/claude-code-playground/)

**ディレクトリ**: `gemini-canvas-slides/`

**主な機能**:
- ✅ **APIキー不要**: 完全にブラウザ上で動作
- ✅ **完全無料**: コスト0円で使用可能
- プロンプトを生成し、ユーザーがGeminiに貼り付けてスライドを作成
- 汎用モードとパターン指定モードの2種類のモードを搭載
- カスタムスタイルの作成・管理機能

詳細は [gemini-canvas-slides/README.md](./gemini-canvas-slides/README.md) をご覧ください。

---

### 研修スライド自動生成ツール

記事やノートの内容から、研修用のスライドを自動生成するWebアプリケーション。

**ディレクトリ**: `slide-generator/`

**主な機能**:
- 記事やノートを入力すると、Claude APIが自動で要約・構造化
- プレゼンテーション資料として見やすいスライドを生成
- 発表者向けメモも自動生成
- **注意**: このプロジェクトはClaude APIキーが必要です

詳細は [slide-generator/README.md](./slide-generator/README.md) をご覧ください。
